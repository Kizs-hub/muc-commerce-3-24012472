## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来做判断的输入信息，比如用户登录频次、消费金额、套餐类型等用户属性与行为数据；标签是想要预测的目标结果，比如在用户流失场景里用户是否会流失。
- 为什么要保留测试集：测试集不参与模型训练，模拟现实里没有的的新用户数据，能检验模型的能力，也能避免模型过拟合或者只适配样本数据却没法应对真实新场景，从而得到可靠的模型效果评估结果。
- 为什么83%准确率仍可能没有用：业务里用户流失占比仅约 16.84%，全部预测用户不流失就能拿到约 83% 的基准准确率，但这种做法显然是单一偏激的，没法对长期运营进行合理预测，对业务没有实际帮助，这类指标不能体现业务价值。
